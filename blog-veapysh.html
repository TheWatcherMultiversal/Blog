Release Veapysh v1.0.0

July 29, 2022
Releases

In the previous version of Veapysh, there were some bugs in the code that affected the functionality of the program and the experience of using it, so I took on the task of updating the code to a better built one.

After reading the code several times for bugs, we released the stable version of Veapysh, a terminal utility that will help you manage certain system tasks with a set of automation scripts.

In this version we have fixed important bugs mentioned above, so we proudly announce that our program is officially ready to be used by all those users who are not very familiar with the Linux terminal.

Bugs fixed in version 1.0.0 of Veapysh

· Correction when entering sudo in the command.

· Error downloading the man page for the command.

· Fix listing active scripts.
