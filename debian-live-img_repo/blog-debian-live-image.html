deb-live-image

August 12, 2023
Releases

deb-live-img is a very useful tool for creating Debian live images. It employs a set of scripts that streamline the use of the lb config and lb build commands.

This tool originated as a personal project, enabling me to craft Debian live images with specific modifications tailored to various environments. Furthermore, it offered the advantage of conducting cleaner and more accurate installations aligned with my desired outcomes.

The documentation and additional information about deb-live-image can be found in the repository on my GitHub profile. You can access it by following the link provided below. I'm also providing the source code so you can utilize this tool.
